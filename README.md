# 🛍️ Nostra - Simple E-commerce Webpage

A clean and responsive e-commerce website built using **HTML, CSS, and JavaScript**.

## 🚀 Project Overview

Nostra is a simple front-end project that showcases:

- Modern UI design
- Product collections
- Contact form with validation
- Responsive layout

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (for contact form validation)

## 📂 Pages Included

- 🏠 Home Page (`index.html`)
- 🛒 Product / Collection Page (`product.html`)
- 📞 Contact Page (`contact.html`)

## ✨ Features

- Responsive Navigation Bar
- Hero Section with images
- New Arrivals & Most Wanted sections
- Newsletter Subscription UI
- Footer with social icons
- ✅ Contact Form Validation using JavaScript
